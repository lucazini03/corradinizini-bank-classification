# corradinizini-bank-classification - Bank Term Deposit Prediction
Machine Learning project based on the Bank Dataset of the Kaggle competition 'Binary Classification with a Bank Dataset' (Playground series - Season 5, Episode 8). The dataset is derived from the classic Bank Marketing dataset and contains customer information and previous campaign results.

## Objective
The goal of this project is to develop a machine learning model that predicts whether a bank customer will subscribe to a term deposit based on demographic and marketing campaign data.

## 📊 Dataset Description
The dataset contains information about bank customers such as:

- Age
- Job
- Marital status
- Balance
- Contact information
- Previous campaign results

Target variable:
- `y` → whether the client subscribed to a term deposit (Yes/No)

The dataset contains hundreds of thousands of rows, allowing robust model training.
